# supplementary code and data for manuscript DESCEND

This folder contains source code of the data analysis results for all the public datasets that are used in the manuscript. Each data analysis folder contains:

1. data: we either include the data file we use in the folder if the dataset is small or point out where the data is downloaded if the dataset is large
2. DESCEND result: we include the descend estimation results used in the paper as .rds files
3. source code in R: we include the code generating the DESCEND result and the code generating our figures in the paper for reproducibility
